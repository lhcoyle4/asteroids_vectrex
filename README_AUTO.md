# Asteroids Vectrex

A WebAssembly port of the classic Asteroids arcade game with Vectrex-style vector graphics.

## Features
- Classic Asteroids gameplay
- Vector-based graphics rendering
- Retro arcade aesthetic
- High score tracking

## Build

```bash
make
```

## Run

Open `index.html` in a web browser, or run the included executable.

## Dependencies
- SDL2
- SDL2_mixer
- Emscripten (for WebAssembly builds)

## Assets
Game assets are located in the `assets/` directory.
